Fettle by thehiflyer

A state machine framework for java

Fettle is an attempt to write a well tested solid simple framework for statemachines in java. I have tried to keep it easy to use, type safe and still flexible. To create a new state machine you can either make a builder object or just create a transition model. TODO: add code examples

Dependencies

Runtime: guava Build time: JUnit Mockachino All libs are checked in

Install

To build: ant all To make a jar: ant all package
